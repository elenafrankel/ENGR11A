Project 1

Learning to Design and Print

Learn More

Designing my Nametag

The first part of the project was to print a nametag. To do this, I:

- Downloaded Prusaslicer and set it up for a Prusa MINI+ 0.4mm nozzle (with Input Shaper)

- Opened the given nametag file using the software

- Edited the file so that the nametag showed my name

- Sliced the design and exported the g-code to a thumb drive

- Filled out the google form to get a print job number

- Set up the 3D printer and learned how to start a print

Printing my Nametag

Starting a print involved:

- Writing the print job number on a sticky note on the printer I chose

- Inserting the USB into the printer

- Finding the design on the user interface

- Loading the filament into the machine

- Adjusting the settings on the printer

- Switching the filament halfway through the print to get a different color for the name

- Removing the print from the printer safely

Challenges and Problem Solving

One challenge I faced during the first part of this project was loading the filament into the printer correctly. Initially, I fed the filament into the tube, and tried to print my design, but nothing happened! I found that I needed to go into the loading settings of the printerâ€™s user interface, and I made sure that the filament was being pulled into the printer. After this, I was able to print the first part of my nametag.

However, I ran into another problem when switching the color of the filament halfway through the printing process. When loading the second filament, the tube which helps feed the filament into the printer fell off. I asked for help, and the TA was able to put it back in place, and I learned to be gentle when loading the filament, and to let the printer pull the filament in, not to push the filament.

Second Print

Once I was finished printing my nametag, I went to thangs.com to find another, more complicated, print. I landed on a hollow sphere with cutouts, held on a pedestal. I opened the file in PrusaSlicer and sliced the design, as done before with the nametag. This time, PrusaSlicer warned me about possible instability in the print, but I ignored this warning to see what the printer was really capable of. I wanted to see where it might have challenges, and in doing so, reveal the physical constraints of what I am able to print on the Prusa MINI+. I proceeded to start the printing process as I did earlier with the nametag, only this time I had a much better idea of what I was doing.
